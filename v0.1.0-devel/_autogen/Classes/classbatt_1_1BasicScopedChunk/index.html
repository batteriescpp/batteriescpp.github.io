    inline void back_up(
    usize n =std::numeric_limits< usize >::max()
)

    Truncates the data chunk by the specified number of bytes, starting at the end.

    For example, BasicScopedChunk::back_up(0) will leave this object unchanged, whereas BasicScopedChunk::back_up(this->size()) will cause the buffer to be re-sized to 0, causing 0 bytes to be consumed when the chunk goes out of scope.
