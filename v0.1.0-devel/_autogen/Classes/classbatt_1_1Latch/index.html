    Detailed Description🔗

    1
2
    template <typename T >
class batt::Latch;

    A write-once, single-value synchronized container.

    Similar to a Future / Promise pair, but Latch has no defined copy/move semantics.

    template <typename... Args>
inline bool set_value(
    Args &&... args
)

    Sets the value, closing the latch.

    Return: true if the Latch was not previously set and this call succeeded in setting its value, or false if the value was previously/concurrently set by another thread.

    args are used to construct a StatusOr, so you can pass an instance of T to set_value, or a Status to indicate an error occurred.
