    void wake_all(Issuer * issuer, ConsumerList & to_wake, Status status)
    Consumes the passed to_wake list, resolving each handler with a freshly issued grant (if status is ok), or the passed error status.

    Create a new State with the specified initial count.
    State(State && that)
    Transfers all waiters and count from that to newly created State.

    Transfer n count to the caller, possibly blocking (if wait_for_resource is true).

    Attempt a fast, non-blocking acquisition of count from this State; returns true iff n was successfully subtracted from this State's count.
