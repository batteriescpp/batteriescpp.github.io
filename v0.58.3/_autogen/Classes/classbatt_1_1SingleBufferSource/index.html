    batt::SingleBufferSource

    batt::SingleBufferSource🔗

    Adapts a single ConstBuffer to be a BufferSource. #include <batteries/async/buffer_source.hpp>

    Public Functions🔗

    Name
    SingleBufferSource() =default
    Constructs a BufferSource with zero bytes (empty).
    SingleBufferSource(const SingleBufferSource & ) =default
    SingleBufferSource is copyable by assignment and move.

    SingleBufferSource is copyable by assignment and move.
    SingleBufferSource(const ConstBuffer & buffer)
    Constructs a SingleBufferSource containing exactly the passed buffer.
    SingleBufferSource(const void * ptr, usize size)
    Constructs a SingleBufferSource containing size bytes, beginning at ptr.

    Constructs a SingleBufferSource containing the characters of the passed string.

    Assigns the buffer source to the passed buffer.

    usize size() const
    Returns the current size of the buffer.
